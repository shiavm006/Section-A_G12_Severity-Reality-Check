## Featured Capstone Projects (showcased across all portfolios)

### Capstone 2 — Severity Reality Check  *(Team G12)*

US road accident severity bias analysis. Disentangled true accident severity from congestion bias across 7.7M-row US Accidents dataset (95K balanced sample). Python ETL + chi-square / Kruskal-Wallis statistical tests + 3 interactive Tableau dashboards.

- **Repo:** [https://github.com/shiavm006/Section-A_G12_Severity-Reality-Check](https://github.com/shiavm006/Section-A_G12_Severity-Reality-Check)
- **Tableau Public:** [https://public.tableau.com/views/RoadAccidentDataofUSA_17773613139680/Dashboard1](https://public.tableau.com/views/RoadAccidentDataofUSA_17773613139680/Dashboard1)
- **Headline finding:** ~50% of public "Severity 4" accidents are congestion artefacts, not real crashes.

### Capstone 1 — Airbnb Reality Check  *(Group G-7)*

Amsterdam Airbnb revenue intelligence — 10,361 listings × 20+ neighbourhoods. KPI-driven dashboard built in Google Sheets with pivot tables.

- **Repo:** [https://github.com/shiavm006/DVA-Capstone](https://github.com/shiavm006/DVA-Capstone)
- **Headline finding:** EUR 200–300 listings achieve 84% occupancy — the strategically optimal price band.
